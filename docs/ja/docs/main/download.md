# ダウンロード管理

## 概要

ダウンロード管理はイラストのダウンロードとノベルのダウンロードをサポートしており、高度なフィルタリング機能があります。

::: danger

V1.6.6 - 1.8.0 のバージョンはバックグラウンド EPUB ダウンロード機能をサポートしていません。

:::

## ダウンロードパスの設定

1. 設定ページに移動する
2. 「ダウンロード設定」オプションを探す
3. 「ダウンロードフォルダを選択」をクリック
4. システムファイルピッカーでイメージを保存したいフォルダを選択する
5. 認可を確認する

## 保存位置

| プラットフォーム                         | 保存位置                       | 認可要件                     | 構成方法 / アクセス方法   |
| ---------------------------------------- | ----------------------------------- | ---------------------------------------------- | -------------------------------------- |
| **iOS**                                  | iPhone/Pixiv-MultiPlatform       | 追加の認可は必要ありません          | Files アプリ経由で直接アクセス可能  |
| **Android**                              | ユーザーが認可したフォルダ              | 設定でダウンロードパスを認可する必要があります    | ギャラリーで表示されます                     |
| **Desktop (Windows/Linux/macOS)**       | ユーザーが指定したフォルダ               | 設定でダウンロードパスを構成する必要があります    | ファイルマネージャーで表示されます                |

::: danger

iOS ユーザーは `My iPhone/Pixiv-MultiPlatform` ディレクトリに表示されている他の非画像ファイルを安全に削除できます。これらは V1.6.6 前のデータファイルであり、現在のバージョンの正常な使用には影響しません。

:::

### 高度な検索 (V1.8.2+)

高度な検索機能により、ダウンロード管理でダウンロードされたコンテンツをフィルタリングおよび検索できます。

#### 検索タイプ

次の 3 つの検索タイプのいずれかを選択できます:

- **すべて**: ダウンロードされたすべてのコンテンツを検索します (イラストとノベルを含む)
- **イラスト**: ダウンロードされたイラストのみを検索します
- **ノベル**: ダウンロードされたノベルのみを検索します

#### キーワード検索

キーワード入力ボックスに検索したい内容を入力してください。 システムは、選択した検索タイプと検索オプションに基づいてマッチします。

#### メタデータの検索

「メタデータの検索」オプションを有効にすると、キーワードはファイル名だけでなく、イラスト自体の情報、以下を含めてマッチできます:

- **タグ**: イラストに関連するタグをマッチさせる
- **イラスト ID**: イラストの一意の識別子をマッチさせる
- **その他のメタデータ**: イラストの他の固有情報をマッチさせる

::: tip

「メタデータの検索」オプションが有効でない場合、検索はファイル名のみをマッチさせます。

:::

## V1.6.6 前のダウンロード管理機能 (アーカイブ)

::: danger

このバージョン以前は保守されていないため、関連する問題は直接閉じられます。

:::

- イメージはアプリの非公開ディレクトリに保存されます
- ダウンロード完了後、ユーザーは「保存」ボタンをクリックして、外部ファイルシステムのイメージにアクセスする必要があります
- ユーザーはダウンロードされたファイルに直接アクセスできません
