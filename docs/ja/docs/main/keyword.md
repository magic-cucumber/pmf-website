# ショートカット

::: danger

Android では、ショートカットサポートが正常に機能するには外部キーボードが必要です

:::

## イラスト和小説ページショートカット

- `R`: 1 回クリックして上部に戻ります、再度クリックして更新します
- `↑`, `↓`: ページをわずかに移動します
- `PgDn`, `PgUp`: ページを大きく移動します

::: info

ラップトップでは、`PgDn` と `PgUp` はそれぞれ `Fn+↓` と `Fn+↑` です

:::

- `←`, `→`: タブ間で移動します。タブ付きのページのみがこのショートカットをサポートしています。

## 画像ブラウザーショートカット

- `Shift`+`←` 1 つの画像を左に移動します
- `Shift`+`→` 1 つの画像を右に移動します
