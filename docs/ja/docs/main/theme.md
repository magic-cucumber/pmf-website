## カスタムテーマ

1. [Material Theme Builder](https://material-foundation.github.io/material-theme-builder/) を開く

   ![image-20250127185957340](./theme.assets/image-20250127185957340.png)

2. 好きな色を設定してください
   ![image-20250127190141529](./theme.assets/image-20250127190141529.png)

3. 右下の `フォントを選択` をクリックしてから、`テーマをエクスポート` をクリックしてください

   ![image-20250127190311349](./theme.assets/image-20250127190311349.png)

4. エクスポートタイプについては、`Material Theme(JSON)` を選択してください

   ![image-20250127190405154](./theme.assets/image-20250127190405154.png)

5. エクスポート後、ソフトウェア設定を開き、テーマインポートをクリックしてください。

   ![image-20250127190521785](./theme.assets/image-20250127190521785.png)

6. その後、テーマは成功しました! ![image-20250127190556868](./theme.assets/image-20250127190556868.png)
