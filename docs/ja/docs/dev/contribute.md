# 機能提案の提出

::: info 注意
`Github アカウント` がない場合は、[Telegram グループ](https://t.me/+n_xsrc1Z590xNTY9) に参加できます
:::

## 提出前の準備

- 最新バージョンの Pixiv-MultiPlatform を使用していることを確認してください。探している機能は既に最新バージョンで実装されている可能性があります。
- 質問する前に、[スマートな質問をする方法](http://www.catb.org/~esr/faqs/smart-questions.html) を確認してください
- 著者の返信にタイムリーに対応できることを確認してください。応答がない問題は長期間保存されて放棄されます。
- GitHub で問題を処理します。 GitHub アカウントがない場合は、最初に登録してください。

## 提出手順

ソフトウェア使用時にバグが発生した場合、または改善提案がある場合は、[バグ報告を提出](https://github.com/magic-cucumber/Pixiv-MultiPlatform/issues/new?template=功能故障.md) をクリックしてバグ問題をアップロードしてください。

ソフトウェア使用時に新しい機能が必要な場合は、[新しい問題](https://github.com/magic-cucumber/Pixiv-MultiPlatform/issues/new?template=功能建议.md) をクリックして機能リクエスト問題をアップロードしてください。
