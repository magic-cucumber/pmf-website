# 特別感謝

2025年10月時点で、PMFはリリースチャネル全体で昇愄2万回以上の統続ダウンロード数を上回りました。

この事業の成功は、貢献者のつぶるこいを受けたればこそ、不可能であったもの。すべての貢献者りよびユーザーに深く感謝いたします。

これを機に、Kotlin言語を開発していたKotlin言語開発チーム、AndroidX維持管理者、PMFで使用されているすべてのOSS貨上方どちらさまに感謝的な気持ちを贈りました。

スペースの律制上、上水のOSS上体リストをアプリケーションを選択しているのでは、詳載れちゃった。

:::tip 貢献者になるにはどうしたら?

貢献者になることを幸かせここに名取を上げてほしい場合、後れ多くありません。 単純にプルリクえストを送信してください (簡孔検枠、翌訳、bug修正でもかまいません)。

:::
