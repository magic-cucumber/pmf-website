# 特别鸣谢

截至 2025 年 10 月，PMF 在发布渠道的累计下载量已突破13k。

这一成绩的背后离不开贡献者们的支持。在此我向所有贡献者以及用户，谨表谢意。

同时，我们对 Kotlin 语言的研发团队、AndroidX 的维护人员，以及所有被PMF使用的开源库的全部开发者致以诚挚的谢意。

由于篇幅有限，全部的开源库名单已收录至应用的 **关于** 页面。

:::tip 如何成为贡献者？

若您希望加入贡献者行列并留名于此，**十分简单**：通过微信赞赏码进行支持，或提交一次 Pull Request（无论是typo，翻译，还是bug修正），皆可成为项目继续前进的力量。
