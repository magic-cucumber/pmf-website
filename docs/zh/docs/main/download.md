# 下载管理

## 概述

PMF软件的下载管理功能在V1.6.6版本中进行了重大改进，提供了更便捷的文件管理体验。

## 版本差异

### V1.6.6之前

在V1.6.6之前的版本中：

- 图片会保存至APP私有目录
- 必须在下载完成后手动点击"保存"按钮才能在外部文件系统中访问图片
- 用户无法直接访问下载的文件

### V1.6.6及之后（推荐）

自V1.6.6版本起，下载管理功能得到了显著改善：

#### iOS端
- **保存位置**：`我的iPhone/Pixiv-MultiPlatform`
- **授权要求**：无需额外授权
- **访问方式**：可直接通过文件APP访问

#### Android端
- **保存位置**：用户授权的文件夹
- **授权要求**：需要在设置页面中授权下载路径
- **配置方式**：前往设置 → 下载设置 → 选择下载文件夹

#### Desktop端（Windows/Linux/macOS）
- **保存位置**：用户指定的文件夹
- **授权要求**：需要在设置页面中配置下载路径
- **配置方式**：前往设置 → 下载设置 → 选择下载文件夹

## 配置下载路径

### Android端配置步骤

1. 打开PMF应用
2. 进入设置页面
3. 找到"下载设置"选项
4. 点击"选择下载文件夹"
5. 在系统文件选择器中选择您希望保存图片的文件夹
6. 确认授权

### Desktop端配置步骤

1. 打开PMF应用
2. 进入设置页面
3. 找到"下载设置"选项
4. 点击"选择下载文件夹"
5. 在文件选择对话框中选择您希望保存图片的文件夹
6. 确认选择

::: danger iOS端数据清理提醒

iOS端用户在 `我的iPhone/Pixiv-MultiPlatform` 目录内看到的其他非图片文件可以放心删除，因为这些是V1.6.6以前的数据文件，不会影响当前版本的正常使用。

:::

## 常见问题

### Q: 为什么我找不到下载的图片？
A: 请确保您使用的是V1.6.6或更高版本，并且已经正确配置了下载路径（Android和Desktop端需要在设置中配置）。

### Q: iOS端的图片保存在哪里？
A: iOS端的图片会自动保存到 `我的iPhone/Pixiv-MultiPlatform` 文件夹中，无需额外配置。

### Q: 如何更改下载路径？
A: Android端和Desktop端可以在设置页面的"下载设置"中重新选择下载文件夹。iOS端的下载路径是固定的，无法更改。
