# 下载管理

## 概述

下载管理支持图片下载和小说下载，且支持高级筛选功能

::: danger

V1.6.6 - 1.8.0之间的版本不支持后台epub下载功能。  

:::


## 配置下载路径

1. 进入设置页面
2. 找到"下载设置"选项
3. 点击"选择下载文件夹"
4. 在系统文件选择器中选择您希望保存图片的文件夹
5. 确认授权

## 保存位置

| 平台                                 | 保存位置                       | 授权要求                     | 配置方式 / 访问方式   |
| ------------------------------------ | ------------------------------ | ---------------------------- | --------------------- |
| **iOS端**                            | 我的iPhone/Pixiv-MultiPlatform | 无需额外授权                 | 可直接通过文件APP访问 |
| **Android端**                        | 用户授权的文件夹               | 需要在设置页面中授权下载路径 | 相册可见              |
| **Desktop端（Windows/Linux/macOS）** | 用户指定的文件夹               | 需要在设置页面中配置下载路径 | 文件管理器可见        |

::: danger

iOS端用户在 `我的iPhone/Pixiv-MultiPlatform` 目录内看到的其他非图片文件可以放心删除，因为这些是V1.6.6以前的数据文件，不会影响当前版本的正常使用。

:::

### 高级搜索(V1.8.2起)

高级搜索功能允许您在下载管理中对已下载的内容进行筛选和查找。

#### 搜索类型

您可以选择以下三种搜索类型之一：

- **全部**：在所有已下载的内容中搜索（包括插画和小说）
- **插画**：仅在已下载的插画中搜索
- **小说**：仅在已下载的小说中搜索

#### 关键词搜索

在关键词输入框中输入您要搜索的内容，系统会根据您选择的搜索类型和搜索选项进行匹配。

#### 搜索元数据

启用"搜索元数据"选项后，关键词不仅可以匹配文件名，还可以匹配插画本身的信息，包括：

- **标签（Tag）**：匹配插画关联的标签
- **插画ID**：匹配插画的唯一标识符
- **其他元数据**：匹配插画的其他内在信息

::: tip

如果不启用"搜索元数据"选项，搜索将仅匹配文件名。

:::


## V1.6.6以前的下载管理功能（归档）

::: danger

此版本及以前的版本不予维护，相关issue会直接close。

:::

- 图片会保存至APP私有目录
- 必须在下载完成后手动点击"保存"按钮才能在外部文件系统中访问图片
- 用户无法直接访问下载的文件
